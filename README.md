# Premium Unixporn Hyprland Rice

A dark premium Hyprland rice for Nobara/Fedora-based Linux.

## Features

- Dynamic Island Waybar
- Premium Rofi launcher
- Custom dashboard with Cava + animated bonsai
- Premium Eww power menu
- Hyprlock lockscreen
- Hypridle idle behaviour
- Rice control center
- Keybind HUD
- Custom ricefetch
- Terminal polish with Kitty, btop, and Cava
- Matching topographic wallpapers

## Main keybinds

| Bind | Action |
|---|---|
| `SUPER + ENTER` | Open Kitty |
| `ALT + SPACE` | App launcher |
| `SUPER + SPACE` | App launcher |
| `SUPER + L` | Power menu |
| `SUPER + SHIFT + L` | Lock screen |
| `SUPER + SHIFT + S` | Showcase mode |
| `SUPER + CTRL + SPACE` | Rice control center |
| `SUPER + F1` | Keybind HUD |
| `PRINT` | Full screenshot |
| `SHIFT + PRINT` | Area screenshot |
| `CTRL + PRINT` | Copy area screenshot |
| `SUPER + PRINT` | Edit screenshot |
